# Alternate 8 LED Blinking Pattern Using AT89C51 Microcontroller

## Description
This project demonstrates an alternate 8 LED blinking pattern (01010101) using the AT89C51 (8051) microcontroller with Embedded C. The LEDs blink alternately in a continuous sequence. The project is developed using Keil uVision and simulated in Proteus.

## Components Used
- AT89C51 Microcontroller
- 8 LEDs

## Software Used
- Keil uVision
- Proteus 8
- Embedded C

## Files
- alternate_led.c – Embedded C source code
- alternate_led.uvproj – Keil project file
- STARTUP.A51 – Startup file
- alternate_led.pdsprj – Proteus project file
- circuit.cng - circuit diagram
